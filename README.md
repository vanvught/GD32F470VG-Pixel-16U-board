#GD32F470 Pixel

Features: 

- 16x 4 Universes pixel output
- 1x SPI Flash ROM for storing the configuration and Art-Net Fail-safe record
- 1x I2C (display)
- 1x UART (programming/debug)

Firmware: [https://github.com/vanvught/GD32F470VG-Pixel-16U](https://github.com/vanvught/GD32F470VG-Pixel-16U)
